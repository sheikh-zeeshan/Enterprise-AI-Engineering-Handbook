# Alpha Car Detailing

This folder contains the handbook's running enterprise example: a nationwide vehicle washing and detailing platform serving walk-in customers, corporate fleets, government departments, insurance companies, and rental companies.

## Planned content

- `docs/` — domain, architecture, ADRs, and operational documentation
- `src/` — .NET 10 reference implementation
- `tests/` — unit, integration, architecture, contract, and end-to-end tests
- `deploy/` — containers, Kubernetes, Azure, CI/CD, and environment assets

The implementation will evolve alongside the handbook chapters.
